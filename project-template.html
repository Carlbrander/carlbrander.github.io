AI-Powered Autonomous Robot

Academic

2024

Project Overview

Developed an autonomous robot capable of navigating complex environments using computer vision and deep learning. The robot can identify objects, avoid obstacles, and perform basic manipulation tasks in real-time.

This project was part of my Master's thesis at ETH Zurich, focusing on the integration of vision-based perception with reinforcement learning for robotic control. The system achieved 85% success rate in object recognition and 92% accuracy in path planning across various test environments.

Technologies Used

  • Python & PyTorch for deep learning models
  • ROS2 for robot control and communication
  • OpenCV for computer vision processing
  • TensorFlow for reinforcement learning
  • Gazebo for simulation and testing
  • C++ for low-level robot control

Key Features

  • Real-time object detection and classification using YOLO v8
  • Autonomous navigation with obstacle avoidance
  • Multi-modal sensor fusion (camera, LiDAR, IMU)
  • Reinforcement learning for adaptive behavior
  • Web-based control interface for remote monitoring
  • Comprehensive logging and data analysis system

Technical Challenges

The main challenge was integrating multiple AI models while maintaining real-time performance. I implemented a custom neural network architecture that combines CNN features with transformer attention mechanisms, achieving 30% faster inference times compared to baseline models.

Results & Impact

The project resulted in a research paper presented at ICRA 2024 and contributed to ongoing research in autonomous robotics at the ETH Robotics Lab. The codebase has been open-sourced and is being used by several research groups worldwide.
